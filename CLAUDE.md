## Project Overview

This is a landing page project for an automation services business specializing in:
- AI chatbot development (WhatsApp, Instagram, Facebook)
- Custom CRM architecture and integration
- Workflow automation solutions

## Site Structure

The site is organized into 7 main sections as defined in `guidelines.txt`:

1. **Hero Section**: Primary value proposition with CTA
2. **Pain Identification**: 3-column grid highlighting automation pain points
3. **Core Solutions**: Feature blocks for chatbots, CRMs, and workflow automation
4. **Lead Magnet**: Free value offering (e.g., downloadable guide)
5. **Case Studies**: Social proof with business results (not technical details)
6. **Process**: 4-step timeline (Discovery → Strategy → Build → Handover)
7. **Final CTA & Footer**: Calendar embed and contact links

## Key Components

- **Persistent Navigation**: Logo, anchor links, "Book a Call" button
- **Floating WhatsApp Button**: Sticky icon linking to automated lead-qualification bot
- **Responsive Design**: Mobile-optimized layout
- **Lead Capture**: Email/WhatsApp opt-in forms throughout

## Design Guidelines

- Focus on business outcomes (time saved, revenue gained) rather than technical implementation
- Use clean, modern aesthetics with high-contrast CTAs
- Include visual assets: dashboard mockups, app connection graphics, bot demo videos
- Case studies should highlight metrics (hours saved, response time reduced)

## Target Audience

Businesses struggling with:
- Slow social media response times
- Manual data entry processes
- Disorganized customer tracking
